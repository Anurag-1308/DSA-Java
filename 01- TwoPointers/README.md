# Two Pointers Pattern
# Description
The Two Pointers pattern uses two indices to solve problems efficiently.

# Common Use Cases
- Pair Sum
- Remove Duplicates
- Sorted Arrays
- Triplet Problems

# Complexity
Most Two Pointers problems improve brute force solutions by reducing nested loops.

Typical Complexity:
- Time: O(n) to O(n²)
- Space: O(1)
