# Sliding Window Pattern
# Description
The Sliding Window pattern is used to efficiently process contiguous subarrays or substrings by maintaining a window and expanding/shrinking it as needed.

# Common Use Cases
- Fixed Size Window
- Variable Size Window
- Longest Substring Problems
- Minimum Window Problems
- Frequency Count Problems
- Subarray Sum/Product Problems

# Complexity
Most Sliding Window problems improve brute force solutions by avoiding repeated work.

Typical Complexity:
- Time: O(n)
- Space: O(1) to O(k)
